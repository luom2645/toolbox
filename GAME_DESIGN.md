# 数字英雄传 - 游戏设计文档

**版本：** 1.0.0  
**类型：** 2048 + RPG 融合  
**平台：** Android / iOS  
**引擎：** Flutter

---

## 🎮 核心玩法

### 基础规则 (2048)
- **4×4 网格**滑动方块
- **相同数字合并**升级 (2→4→8→16→...→2048)
- **每次滑动**新方块生成 (2 或 4)
- **游戏结束**：网格填满且无法合并

### RPG 创新元素

#### 1. 英雄角色系统
每个数字对应一个英雄角色：

| 数字 | 英雄等级 | 角色名称 | 外观 |
|------|----------|----------|------|
| 2 | LV.1 | 新手剑士 | 青铜盔甲 |
| 4 | LV.2 | 见习骑士 | 铁甲 |
| 8 | LV.3 | 精英战士 | 钢甲 |
| 16 | LV.4 | 荣耀骑士 | 金色盔甲 |
| 32 | LV.5 | 钢铁领主 | 发光钢甲 |
| 64 | LV.6 | 传奇英雄 | 紫色传奇甲 |
| 128 | LV.7 | 史诗王者 | 橙色史诗甲 |
| 256 | LV.8 | 神话战神 | 红色神话甲 |
| 512 | LV.9 | 不朽天神 | 金色天神甲 |
| 1024 | LV.10 | 创世神祇 | 七彩神光 |
| 2048 | LV.MAX | 数字之主 | 终极形态 |

#### 2. 剧情模式
- **每合成一个新数字**解锁一段剧情
- **剧情章节**：对应数字等级
- **故事背景**：数字王国的英雄收集与成长

#### 3. 技能系统
特殊方块=技能卡：

| 技能 | 效果 | 获取方式 |
|------|------|----------|
| 💣 炸弹 | 消除一个方块 | 达到 64 解锁 |
| ❄️ 冻结 | 冻结一回合不生成新块 | 达到 128 解锁 |
| 🔄 交换 | 交换两个方块位置 | 达到 256 解锁 |
| ✨ 复制 | 复制一个方块 | 达到 512 解锁 |
| ⚡ 闪电 | 消除一行或一列 | 达到 1024 解锁 |

#### 4. 装备系统
- **头盔**：提升合并得分
- **武器**：增加技能威力
- **盔甲**：减少失误惩罚
- **饰品**：特殊效果 (幸运值等)

#### 5. 关卡系统
| 模式 | 说明 |
|------|------|
| 📖 剧情模式 | 主线故事推进，解锁角色 |
| ♾️ 无尽模式 | 挑战最高分 |
| 📅 每日挑战 | 特殊规则每日任务 |
| ⚔️  Boss 战 | 特定数字挑战 |

## 🎨 视觉设计

### 色彩方案
```
数字 2-32:    蓝色系 (新手→精英)
数字 64-256:  紫色系 (传奇→神话)
数字 512-2048: 金色/红色系 (天神→终极)
```

### 动画效果
- **滑动动画**：方块移动平滑过渡
- **合并动画**：缩放 + 闪光效果
- **升级动画**：粒子爆发 + 升级文字
- **连击动画**：Combo 计数 + 屏幕震动

### UI 布局
```
┌─────────────────────────┐
│  分数  │  最高分  │ 菜单 │
├─────────────────────────┤
│                         │
│    4×4 游戏网格          │
│    (带英雄角色显示)      │
│                         │
├─────────────────────────┤
│  技能栏  │  当前剧情     │
└─────────────────────────┘
```

## 📊 数值设计

### 方块生成概率
- **普通方块 (2)**: 90%
- **普通方块 (4)**: 10%
- **技能方块**: 特殊关卡

### 得分规则
- **合并得分**：合并后的数字值
- **连击加成**：连续合并×倍率
- **剧情奖励**：解锁新数字奖励
