# 数字英雄传 - 2048 RPG

> 经典 2048 玩法 + RPG 角色养成，打造你的数字英雄战队！

## 🎮 游戏特色

### 核心玩法
- **经典 2048**: 滑动合并相同数字
- **英雄收集**: 每个数字=一个英雄角色
- **剧情模式**: 解锁数字推进故事
- **技能系统**: 特殊方块=技能卡

### 英雄角色
| 数字 | 英雄 | 数字 | 英雄 |
|------|------|------|------|
| 2 | 新手剑士 | 64 | 传奇英雄 |
| 4 | 见习骑士 | 128 | 史诗王者 |
| 8 | 精英战士 | 256 | 神话战神 |
| 16 | 荣耀骑士 | 512 | 不朽天神 |
| 32 | 钢铁领主 | 1024 | 创世神祇 |
| - | - | 2048 | **数字之主** |

## 🚀 快速开始

### 环境要求
- Flutter SDK >= 3.0.0
- Android SDK >= 21
- JDK 1.8+

### 安装运行
```bash
cd flutter_toolbox_app
flutter pub get
flutter run
```

### 构建 APK
```bash
flutter build apk --release
```
